Висновки

У результаті виконання роботи ми закріпили знання та практичні навички щодо створення HTML-документів, використання тегів та атрибутів, а також роботи з системами контролю версій Git та платформою GitHub.

Основні результати та досягнення:

  • Ознайомлення з тегами та атрибутами HTML: вивчено та застосовано заголовки, параграфи, списки, посилання, зображення, таблиці, форми та їхні атрибути (class, id, src, alt, href, type, name, value, placeholder, required).
  • Структуризація HTML-документу: створено логічну та семантичну структуру з шапкою, основним контентом і футером; організовано блоки для таблиць, форм та зображень для зручності відображення.
  • Практична робота з Git та GitHub: створено репозиторії, виконано коміти, відправку змін на віддалений репозиторій; закріплено принципи ведення історії змін і спільної роботи над проектом.
  • Розробка шаблону звітного HTML-документу: створено універсальний шаблон для оформлення результатів практичних робіт з блоками коду, таблиць, форм, зображень та пояснень атрибутів.

Таким чином, виконана робота дозволила набути практичні навички роботи з HTML, таблицями, формами, зображеннями та списками, а також освоїти базові операції з Git і GitHub. Розроблений шаблон забезпечує зручне та структуроване відображення результатів практичних робіт і може служити основою для подальшого розвитку веб-застосунків.
